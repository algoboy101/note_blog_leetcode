## 题目描述

<p>给定一个由 0 和 1 组成的矩阵，找出每个元素到最近的 0 的距离。</p>

<p>两个相邻元素间的距离为 1 。</p>

<p><strong>示例 1: </strong><br />
输入:</p>

<pre>
0 0 0
0 1 0
0 0 0
</pre>

<p>输出:</p>

<pre>
0 0 0
0 1 0
0 0 0
</pre>

<p><strong>示例 2: </strong><br />
输入:</p>

<pre>
0 0 0
0 1 0
1 1 1
</pre>

<p>输出:</p>

<pre>
0 0 0
0 1 0
1 2 1
</pre>

<p><strong>注意:</strong></p>

<ol>
	<li>给定矩阵的元素个数不超过 10000。</li>
	<li>给定矩阵中至少有一个元素是 0。</li>
	<li>矩阵中的元素只在四个方向上相邻: 上、下、左、右。</li>
</ol>
