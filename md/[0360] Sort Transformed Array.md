> 
> 要注意，返回的这个数组必须按照 升序排列，并且我们所期望的解法时间复杂度为 O(n)。 
> 
> 示例 1： 
> 
> 输入: nums = [-4,-2,2,4], a = 1, b = 3, c = 5

>
